1. The code has been taken from https://www.tensorflow.org/tutorials/text/image_captioning
2. Datasets can be downloaded from http://images.cocodataset.org/annotations/annotations_trainval2014.zip and http://images.cocodataset.org/zips/train2014.zip
3. To run the code make sure to keep the downloaded folders in the same directory of file.
The directory should be like this:
--annotations
--train204
--NASnet_model.ipynb
--RESNET_model.ipynb
--VGG_model.ipynb
